# Landing Page Cua Tuyết — Hải Sản Hoàng Gia

Trang web giới thiệu và tư vấn đặt trước sản phẩm Cua Tuyết Canada cao cấp phục vụ tại hệ thống Hải Sản Hoàng Gia.

## 🚀 Giới thiệu
Dự án được xây dựng với mục tiêu cung cấp trải nghiệm số xa xỉ (Quiet Luxury), tối ưu chuyển đổi và cung cấp thông tin chi tiết về sản phẩm Cua Tuyết (xuất xứ Canada) bao gồm combo, cách chế biến, chính sách cam kết chất lượng 1 đổi 1 và câu hỏi thường gặp.

## 🛠️ Công nghệ sử dụng
- **Cốt lõi**: HTML5, Vanilla JavaScript.
- **Giao diện**: Vanilla CSS (tương thích tốt với các thiết bị di động và máy tính).
- **Thiết kế**: Phong cách tối giản sang trọng (Midnight Ocean palette: Navy, Gold, Warm Ivory).

## 📂 Cấu trúc thư mục
```text
├── img/                  # Thư mục chứa hình ảnh sản phẩm cinematic
├── index.html            # File cấu trúc giao diện chính
├── index.css             # File định nghĩa phong cách CSS & Design Tokens
├── main.js               # File xử lý tương tác logic & tracking
├── cua_tuyet.md          # Tài liệu quy trình phát triển & nội dung sản phẩm
└── README.md             # Tài liệu hướng dẫn sử dụng dự án
```

## 💻 Cách chạy dự án dưới local
Để chạy thử giao diện trên trình duyệt cục bộ, bạn có thể sử dụng công cụ `http-server` của Node.js:

```bash
npx http-server -p 8080 -c-1
```

Sau đó mở trình duyệt và truy cập: [http://127.0.0.1:8080](http://127.0.0.1:8080)
